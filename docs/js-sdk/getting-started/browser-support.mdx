We use a number of modern browser technologies to run this SDK in the client, so **it is unsupported for all versions of
Internet Explorer and a few less-used mobile browsers**.

:::info
You can see a matrix of all of the technologies used, together with
browser support at [caniuse.com](https://caniuse.com/wasm,webworkers,cryptography,typedarrays,fetch).
:::
