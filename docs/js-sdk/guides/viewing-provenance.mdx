### Initializing the SDK

The way that you import `wasmSrc` and `workerSrc` varies depending on the build system you use. For more information, see [Quick start](../getting-started/quick-start#bringing-in-the-library).

### Determining whether manifest data exists

To determine whether an asset has associated manifest data, test whether [`c2pa.read`](../../js-sdk/api/c2pa.c2pa#methods) returns a [`manifestStore`](../../js-sdk/api/c2pa.c2pareadresult.manifeststore) that is truthy.
